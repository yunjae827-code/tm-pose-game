# 게임 규칙

## 🎮 게임 제목
**Fruit Catcher (과일 받기 게임)**

## 📝 게임 설명
하늘에서 떨어지는 맛있는 과일들을 바구니로 받아주세요!
폭탄을 피하고 사과와 오렌지를 많이 받아 최고 점수에 도전하세요.
60초 동안 얼마나 많은 점수를 얻을 수 있을까요?

## 🎯 게임 목표
- 제한 시간(60초) 동안 최대한 많은 점수 획득
- 폭탄을 피하여 생존 (폭탄을 받으면 즉시 종료)

## 🕹️ 게임 규칙

### 기본 조작
- **왼쪽 기울이기 (Left)**: 바구니가 왼쪽으로 이동합니다.
- **오른쪽 기울이기 (Right)**: 바구니가 오른쪽으로 이동합니다.
- **정면 (Center)**: 바구니가 제자리에 멈춥니다.

### 아이템 및 점수
| 아이템 | 설명 | 점수 |
|---|---|---|
| 🍎 **사과** | 기본적인 과일입니다. | +100점 |
| 🍊 **오렌지** | 조금 빠르게 떨어집니다. | +200점 |
| 💎 **보석** | 가끔 나타나는 보너스 아이템! | +500점 |
| 💣 **폭탄** | 받으면 **게임 오버**! | - |

## 🎨 필요한 포즈 (Teachable Machine)
1. **Left**: 몸이나 고개를 왼쪽으로 기울인 자세
2. **Right**: 몸이나 고개를 오른쪽으로 기울인 자세
3. **Center**: 정면을 보고 있는 평소 자세

## ⏱️ 게임 플레이
- **시간 제한**: 60초
- **레벨 시스템**:
  - 시간이 지날수록 과일이 떨어지는 속도가 빨라집니다.
  - 점수가 높아지면 폭탄이 더 자주 등장합니다.

## 💡 개발 가이드
- **속도 조절**: gameEngine.js에서 레벨에 따른 속도 계수를 조정하세요.
- **충돌 처리**: 바구니와 아이템의 좌표 거리를 계산하여 충돌을 감지합니다.
